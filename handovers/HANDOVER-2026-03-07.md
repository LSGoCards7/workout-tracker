## Key Decisions
- **Removed `getDayName()`**: Was only called from `renderHome()` (which no longer uses it). Dead code removed rather than left orphaned.
- **`renderHome()` completed-today logic**: Tracks the highest DAYS_ORDER index among completed workouts today (handles multiple-workouts-same-day). Shows both "COMPLETED TODAY" card with continue button AND "NEXT UP" card if the next workout differs.
- **`stopRest(natural)` parameter pattern**: Rather than adding a separate function, added an optional parameter. Only the interval callback passes `true` — all manual dismissals (skip, goBack, setDifficulty, startRest cleanup) pass nothing/falsy.
- **No `getDayName()` replacement in renderHome**: The "TODAY — SAT" label was removed entirely. The today card now shows "COMPLETED TODAY" or "NEXT UP" labels instead of day-of-week.
